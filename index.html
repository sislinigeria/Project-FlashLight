Programming with JavaScript

You can use JavaScript to change the values in an attribute

In this practical, we will use JavaScript to change the value of the src (source) attribute of an image.

First we will make sure we have the images of a bulb that is on and also a bulb that is off. Then we can use programming, to tell the browser to change image source (src) when ever we clicked on each of the buttons.

This also means we will need two button tags that will indicate ON and another for OFF

This is just a bit of what JavaScript can do.
